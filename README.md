## 🚀 如何設置 (Setup Instructions)

### 1. 專案概述 (Project Overview)
本專案包含鴻宴國際新金專業服務的官方網站以及合作品牌 NaNa 髮藝的展示頁面。這是一個基於 HTML/CSS 的靜態網站，支援 Jekyll 部署。

### 2. 系統需求 (System Requirements)
- 瀏覽器：Chrome、Firefox、Safari 或 Edge
- 本地開發（可選）：Python 3.x 或任何網頁伺服器
- 部署：GitHub Pages (已配置)

### 3. 快速開始 (Quick Start)

#### 方法一：直接瀏覽
1. 直接開啟 `index.html` 檔案即可在瀏覽器中查看網站

#### 方法二：本地伺服器
```bash
# 複製專案
git clone https://github.com/love1120520/hongyan-official.git
cd hongyan-official

# 使用 Python 啟動本地伺服器
python3 -m http.server 8000

# 在瀏覽器中開啟
# http://localhost:8000
```

#### 方法三：使用其他伺服器
```bash
# Node.js 使用者
npx http-server

# PHP 使用者
php -S localhost:8000
```

### 4. 專案結構 (Project Structure)

### 5. 部署說明 (Deployment)
- 本專案已配置 GitHub Pages 自動部署
- 推送到 `main` 分支後會自動觸發 Jekyll 建置
- 網站將自動發佈到 GitHub Pages

### 6. 自定義修改 (Customization)
如需修改網站內容：
1. 編輯 `index.html` 更改 NaNa 髮藝頁面內容
2. 編輯 `README.md` 更新專案說明
3. 提交變更後推送至 GitHub

### 7. 驗證設置 (Verify Setup)
設置完成後，您應該能看到：
- NaNa 髮藝主標題
- 服務項目：男士漸層剪 × 油頭造型 × 染燙護理
- 完整的聯絡資訊（LINE、信箱、Instagram、TikTok）
- 美髮沙龍背景圖片

### 8. 疑難排解 (Troubleshooting)
- **頁面無法顯示**：確認瀏覽器支援 HTML5
- **圖片載入失敗**：檢查網路連線，背景使用 Unsplash API
- **本地伺服器問題**：確認已安裝 Python/Node.js/PHP
- **中文顯示異常**：確認瀏覽器編碼設定為 UTF-8
